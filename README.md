# AI Customer Service Software

## Overview
This project implements an AI-powered customer service tool that assists in automating responses to customer inquiries.

## Requirements
- Python 3.8+
- Flask
- TensorFlow
- scikit-learn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Emo265/AI-customer-service-software.git
   cd AI-customer-service-software
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. To run the application:
   ```bash
   python app.py
   ```
